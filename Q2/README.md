# Question 2

## Command 1

```bash
gcc q2.c -o q2
```

Explanation:
This command compiles the C source code and creates an executable file named `q2`.

---

## Command 2

```bash
./q2
```

Explanation:
This command runs the program. It creates a child process, monitors its execution, terminates it using a signal, and waits for it to finish.

---

## Command 3

```bash
./q2 > output.txt
```

Explanation:
This command redirects the program output into a text file.

---

## Command 4

```bash
cat output.txt
```

Explanation:
This command displays the contents of the output file.
