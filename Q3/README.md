# Question 3

## Command 1

```bash
nano q3.c
```

Explanation:
This command creates and edits the C source file.

---

## Command 2

```bash
gcc q3.c -o q3
```

Explanation:
This command compiles the C program and creates an executable named `q3`.

---

## Command 3

```bash
./q3
```

Explanation:
This command executes the program, demonstrating the use of `open()`, `write()`, `read()`, `lseek()`, and `close()`.

---

## Command 4

```bash
./q3 > output.txt
```

Explanation:
This command redirects the program output into `output.txt`.

---

## Command 5

```bash
cat output.txt
```

Explanation:
This command displays the contents of the output file.

---

## Command 6

```bash
cat employee.txt
```

Explanation:
This command displays the employee records stored in the file.
