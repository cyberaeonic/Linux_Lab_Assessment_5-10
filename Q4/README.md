# Question 4

## Command 1

```bash
nano sample.log
```

Explanation:
This command creates a sample log file for testing.

---

## Command 2

```bash
tail -f sample.log
```

Explanation:
This command continuously monitors the log file and displays new entries as they are added.

---

## Command 3

```bash
grep "ERROR" sample.log
```

Explanation:
This command filters the log file and displays only entries containing "ERROR".

---

## Command 4

```bash
grep "ERROR" sample.log > error_report.txt
```

Explanation:
This command saves all error entries into `error_report.txt`.

---

## Command 5

```bash
cat error_report.txt
```

Explanation:
This command displays the contents of the generated error report.
